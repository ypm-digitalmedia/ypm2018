-----------------
Fine Image Upload
-----------------

Description
-----------
This module add a new field widget ('Fine image upload') to the image field.
It makes standard image upload field more user-friendly and simple to use
by showing only necessary settings and adding more convenient image preview
feature, along with details of uploaded images.

Requirements
------------
1) Optional core Image.

2) From version 8.x-2.x - Magnific-popup library. Download Magnific-popup
   (https://github.com/dimsemenov/Magnific-Popup) library and place it in
   the libraries folder (/libraries).

Installation
------------
1) Install the module as usual.

2) Download Magnific-popup (https://github.com/dimsemenov/Magnific-Popup)
   library and place it in the libraries folder (/libraries).

3) Switch image field widget to Fine Image Upload. This can be done at
   "Manage form display" tab of one of yours Content types.
